docker ps : Melihat daftar container yang sedang aktif/berjalan.
docker ps -a : Melihat seluruh container (yang aktif maupun yang sudah mati).
docker images : Melihat daftar image yang sudah terunduh di sistem.
docker stats : Memantau penggunaan CPU, RAM, dan network container secara real-time.
docker run -d --name nama_container -p port_host:port_container nama_image : Membuat dan menjalankan container baru di background (detached mode) dengan nama dan port tertentu.
docker start nama_container : Menyalakan kembali container yang sedang berhenti/stop.
docker stop nama_container : Menghentikan container yang sedang berjalan.
docker restart nama_container : Merestart/memuat ulang container yang sedang berjalan.
docker rm nama_container : Menghapus container yang sudah dalam posisi mati.
docker rm -f nama_container : Memaksa hapus container yang posisinya masih berjalan aktif.
docker logs nama_container : Melihat riwayat catatan/log output dari aplikasi di dalam container.
docker logs -f nama_container : Melihat log output container secara real-time (terus diperbarui).
docker exec -it nama_container bash : Masuk dan mengakses terminal di dalam container yang sedang berjalan.
docker pull nama_image : Mengunduh image dari Docker Hub ke penyimpanan lokal tanpa langsung menjalankannya.
docker rmi nama_image : Menghapus image dari penyimpanan lokal.
docker build -t nama_image_baru . : Membuat image kustom sendiri berdasarkan instruksi dari file Dockerfile di folder saat ini.
docker container prune : Menghapus semua container yang posisinya sudah mati sekaligus.
docker image prune : Menghapus image yang tidak terpakai atau tidak memiliki nama (dangling images).
docker system prune -a --volumes : Menghapus total seluruh container mati, image tidak terpakai, cache, dan volume untuk mengosongkan ruang disk.
docker compose up -d : Menjalankan banyak container sekaligus di background berdasarkan konfigurasi file docker-compose.yml.
docker compose down : Mematikan dan langsung menghapus seluruh susunan container yang dijalankan lewat Docker Compose.
docker compose logs -f : Memantau log output dari seluruh container di Docker Compose secara bersamaan dan real-time.
